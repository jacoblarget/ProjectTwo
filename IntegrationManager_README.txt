Instructions to Build, Run and Test your Project:
-------------------------------------------------
To run the project, use the command "make" to run the program (it should automatically compile).
Then follow the provided Frontend interface to interact with the pokedex.

To compile the project, use the command "make compile" to check proper compilation.

To test the project, use the command "make test" to run the provided JUnit test suites.
To test a particular suite, use the following commands:
Backend: "make testBackend"
Data: "make testData"
Frontend: "make testFrontend"
